# DockerCompose项目推荐

[composerize](https://www.composerize.com/) - docker run转换成docker-compose.yml文件。  
## docker安装脚本-使用阿里云镜像
```sh
curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun
```
